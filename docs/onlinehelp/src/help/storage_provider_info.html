Storage provider information

You must configure a default storage provider for SkyDRM RMS. SkyDRM RMS supports Amazon S3, OneDrive for Business, and local drive as default storage providers. This section describes the storage provider-related information you must have on hand before attempting to install SkyDRM RMS.
Table 1. Amazon S3 storage provider information
Component Value
Amazon S3 Access Key ID The access key ID.
Amazon S3 Secret Access Key The secret access key.
MySpace storage path The storage path where you want files uploaded to MySpace to go to. For example: Documents/drm-mydrive
Projects storage path The storage path where you want files uploaded to Projects to go to. For example: Documents/drm-projects
Table 2. OneDrive for Business storage provider information
Component Value
OneDrive for Business user name Obtain the OneDrive for Business user name.
OneDrive for Business password Obtain the OneDrive for Business password.
OneDrive for Business site URL Obtain the URL.
MySpace storage path The storage path where you want files uploaded to MySpace to go to. For example: Documents/drm-mydrive
Projects storage path The storage path where you want files uploaded to Projects to go to. For example: Documents/drm-projects
Table 3. Local drive storage information
Component Value
Customized repository root path The storage path where you want files to go to. For example: /mnt/shared_repo_path/
MySpace storage path The storage path where you want files uploaded to MySpace to go to. For example: Documents/drm-mydrive

If you want to add a customized path for local storage, you need to edit the compose file to add a new line.
Projects storage path The storage path where you want files uploaded to Projects to go to. For example: Documents/drm-projects
