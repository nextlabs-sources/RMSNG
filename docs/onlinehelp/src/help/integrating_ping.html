Integrating PingOne

PingOne is a single sign-on (SSO) solution for one-click access to SaaS, legacy and custom web applications. SkyDRM RMS can integrate with PingOne, enabling you to log in to SkyDRM RMS using PingOne credentials. This topic describes how to create an PingOne app, which binds PingOne to SkyDRM RMS. You can then add the PingOne information to admin.properties file which enables login with PingOne credentials.
  1. Log in to PingOne as an administrator.
    The URL is https://admin.pingone.com/web-portal/login.
  2. On the PingOne Dashboard, click Applications.
  3. Click on the SAML tab.
  4. Select Add Application > New SAML Application.
    The Application Details pane appears.
  5. Type the application name.
  6. Type the application description.
  7. From the Category drop-down list, select Collaboration.
  8. Click Continue to Next Step.
    The Application Configuration pane appears.
  9. From the Signing Certificate drop-down list, select PingOne Account Origination Certificate.
  10. In the Assertion Consumer Service (ACS) box, type the ACS URL.
    For example, https://<hostname>:8444/rms/SAML/AuthFinish.
  11. In the Entity ID box, type the entity ID.
    For example, https://<hostname>:8444/rms/SAML/Metadata.
  12. Click Continue to Next Step.
    The SSO Attribute Mapping pane appears.
  13. Click Add new attribute.
  14. Map the necessary application provider (AP) attributes to attributes used by your identity provider (IdP).
    For example, map the application attribute email to Email.
  15. Click Save & Publish.
    The Review Setup pane appears.
  16. Next to SAML Metadata, click Download.
  17. Open the downloaded file.
  18. Copy and save the PingOne-related parameters for use during SkyDRM RMS installation.
    1. Copy the ACS URL to spAcsUrl.
    2. Copy the Entity ID to the spEntityId.
    3. Copy the entityID from the downloaded file to idpEntityId.
    4. Copy the SSO URL to idpSsoUrl.
    Sample configuration
    web.idp.saml.3.attributes={\"name":"pingone",\
				"spEntityId":"https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlMetadata",\
				"spAcsUrl":"https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlAuthFinish",\
				"idpEntityId":"https://pingone.com/idp/<yourcompany>",\
				"idpSsoUrl":"https://sso.connect.pingidentity.com/sso/idp/SSO.saml2?idpid=11799c11-b51e-434b-b411",\
				"spNameIdFormat":"",\"signAlgo":"",\"authNContext":"",\"buttonText":"pingone",\
				"idpX509Cert":"MIIDWjCCAkKgAwIBAgIGAWBDex"\}
