Using a SkyDRM project

A SkyDRM project enables you to collaborate with a group of users, on one or more rights-protected documents.

Important: You can only create a project if your SkyDRM administrator grants you the right to create a project.

You can define a SkyDRM project account, specify the members of the project, and share one or more documents with the project members. Project members can see the documents in the project, member activities on the documents, and decide how the documents should be shared.
