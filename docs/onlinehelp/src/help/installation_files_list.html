Installation file list

This section lists the installation files you need to download from Amazon Web Services or NextLabs Customer Portal. Contact your NextLabs professional service personnel to obtain access.

Compulsory files

Table 1. Compulsory files
File type File name Content
Docker image RightsManagementServer-DB-DockerImage-*.tar The database Docker image
RightsManagementServer-RMS-DockerImage-*.tar The server Docker image.
RightsManagementServer-Router-DockerImage-*.tar The router Docker image.
RightsManagementServer-Viewer-DockerImage-*.tar The viewer Docker image.
RightsManagementServer-CacheServer-DockerImage-*.tar The distributed cache server Docker image.
RightsManagementServer-MessageQueue-DockerImage-*.tar The notification server Docker image.
SkyDRM setup archive The SkyDRM setup archive, SkyDRM-setup-10-*.zip The Docker compose file, docker-compose-xxxx.yml.
The shell script, setup.sh.

Non-compulsory files

You may need to download one or more of the following viewer archives, depending on your environment. The default data directory you can put the files in is /opt/nextlabs/rms/shared/viewerPackage.
  • RightManagementServer-DocViewer-10.*.zip
  • RightsManagementServer-SAPViewer-10.*.zip
  • RightsManagementServer-CADViewer-10.*.zip
