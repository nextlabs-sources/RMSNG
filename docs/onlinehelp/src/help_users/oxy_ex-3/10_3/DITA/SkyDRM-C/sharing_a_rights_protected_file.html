Sharing a rights-protected file

You can share a rights-protected file from your local or network drive, MyDrive, or an external repository by email. However, if you are using a company account, you cannot share rights-protected files on external repositories.

  1. On the Home screen, tap MyDrive.
  2. Tap the menu icon next to the file you want to share, and tap Share.
  3. On the Home screen, click Share a protected file.
    Alternatively, you can navigate to the file using the MySpace navigation pane, and click the Share File button.
    The Share a protected file pane appears.
  4. Select a repository from the drop-down list.
    The list of files in the repository appears.
    Alternatively, you can select a file from your local drive.
  5. Click a file to select it.
  6. Click Proceed.
  7. In the Protect using Digital Rights dialog box, select the slider of the rights protection type you want to apply to the file.
    • View: Enables users to view a rights-protected file. This is selected by default and cannot be disabled.
    • Print: Enables users to print a rights-protected file.
    • Share: Enables a user to share the rights-protected file through an email link.
    • Save: Enables users to save a rights-protected file.
    • Watermark: Displays a watermark on a rights-protected file.
    • Validity: defines the expiration of the rights protection for the file.
  8. In the Share the file with box, type the email address of the person you want to share the rights protected file with.
  9. Click Share a protected file.
    A confirmation pane appears, with the rights protection details.
  10. Click OK.
    The file is now available in MyVault.
