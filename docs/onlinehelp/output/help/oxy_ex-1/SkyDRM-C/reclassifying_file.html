Reclassifying a file

After protecting a file, you may find that the file classification is incorrect. When this happens, as a project administrator, you can reclassify the protected file without having to re-upload the file. Only a project administrator can reclassify a file.
  1. On the Home screen, click on a project that you are a member of.
    The project page appears.
  2. On the navigation bar, click Files.
    The list of files in the project appears.
  3. Hover your cursor over the menu icon .
    The available icons appear.
  4. Click Reclassify .
    The Modify rights pane appears.
  5. Select the correct document classification.
    You can click the correct classification, or select a classification from the drop-down list.
  6. Click Save.
    A message appears, informing you that the reclassification is successful.
