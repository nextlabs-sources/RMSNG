Integrating OneLogin

OneLogin is a unified access management platform that provides secure single sign-on, multi-factor authentication, directory integration. SkyDRM RMS can integrate with OneLogin, enabling you to log in to SkyDRM RMS using OneLogin credentials. This topic describes how to create an OneLogin app, which binds OneLogin to SkyDRM RMS. You can then add the OneLogin information to admin.properties file which enables login with OneLogin credentials.
  1. Log in to OneLogin as an administrator.
    The URL is https://<hostname>.onelogin.com/admin.
  2. On the OneLogin Administration page, select Apps > Add Apps.
  3. In the Find Applications page search bar, type SAML Test Connector.
  4. Click SAML Test Connector (IdP w/attr).
    The Configuration page appears.
  5. In the Display Name box, type a name for the new app.
  6. Click the Visible in portal slider.
  7. Click the SAVE.
    The Info tab appears.
  8. Click the Configuration tab.
  9. In the Audience box, type https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlMetadata.
  10. In the Recipient box, type https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlAuthFinish.
  11. In the ACS (Consumer) URL Validator box, type https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlAuthFinish.
  12. In the ACS (Consumer) URL box, type https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlAuthFinish.
  13. Click the Parameters tab.
  14. Click Add parameter.
  15. In the New Field dialog box, type company.
  16. Select the Include in SAML assertion check box.
  17. Click Save.
  18. From the Value drop-down list, select Company.
  19. Click Save.
  20. Click Add parameter.
  21. In the New Field dialog box, type department.
  22. Select the Include in SAML assertion check box.
  23. Click Save.
  24. From the Value drop-down list, select Department.
  25. Click Add parameter.
  26. In the New Field dialog box, type email.
  27. Select the Include in SAML assertion check box.
  28. Click Save.
  29. From the Value drop-down list, select Email.
  30. Click Add parameter.
  31. In the New Field dialog box, type title.
  32. Select the Include in SAML assertion check box.
  33. Click Save.
  34. From the Value drop-down list, select Title.
  35. Click Save.
  36. Click the SSO tab.
  37. Copy and save the OneLogin-related parameters for use during SkyDRM RMS installation.
    1. Copy the Audience URL to the spEntityId.
    2. Copy the Recipient URL to the spAcsUrl.
    3. Click the SSO tab.
    4. Click View Setup Instructions.
    5. Copy the Issuer URL to idpEntityId.
    6. Copy the SAML 2.0 Endpoint (HTTP) to idpSsoUrl.
    7. Copy the X509 Certificate to idpX509Cert.
    Sample configuration
    web.idp.saml.4.attributes={\
				"name":"onelogin",\
				"spEntityId":"https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlMetadata",\
				"spAcsUrl":"https://<hostname>:8444/rms/IdpManager/SamlAuth/SamlAuthFinish",\
				"idpEntityId":"https://app.onelogin.com/saml/metadata/acd53d2b-2e46-49ce-879d-a7c1382f74cb",\
				"idpSsoUrl":"https://<yourcompany.onelogin.com/trust/saml2/http-post/sso/804066",\
				"spNameIdFormat":"",\"signAlgo":"",\"authNContext":"",\"buttonText":"onelogin Login",\
				"idpX509Cert":"MIIEGjCCAwKgAwIBAgIUBuB+qLDDzU4bWvNyYGw4AskNi3UwDQYJKoZIhvcNAQEFBQAwW11"\}
