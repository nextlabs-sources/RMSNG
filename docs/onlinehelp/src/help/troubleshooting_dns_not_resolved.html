Cannot access SkyDRM RMS website: DNS not resolving

Condition

After installation is complete, you should be able to access the SkyDRM RMS website using the URL https://hostname/rms. However, the following error message appears instead: This site cannot provide a secure connection. <hostname> sent an invalid response. Try running Windows Network Diagnostics. ERR_SSL_PROTOCOL_ERROR.

Cause

The Domain Name System (DNS) is not resolved.

Remedy

Edit the hosts file to add the hostname and IP address.

  1. Log in to your system as a user with sudo privileges.
  2. Navigate to the hosts file in the /etc path by typing the following command. 
    cd /etc
  3. Press Enter on your keyboard.
  4. Type the following command. 
    vi hosts
  5. Press i on your keyboard.
  6. Type the IP address and hostname in the following format: <IP address> <hostname>.
    For example:
    10.10.10.10 hostname.nextlabs.com
  7. Press Esc on your keyboard.
  8. Type the following command. 
    wq!
  9. Press Enter on your keyboard.
