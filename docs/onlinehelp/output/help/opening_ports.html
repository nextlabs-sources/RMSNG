Opening ports

This section describes the commands you must run to open the necessary ports for the message queue and cache service.
  1. Run these commands to open the ports necessary for the message queue. 
    $ sudo firewall-cmd --permanent --add-port=5672/tcp

$ sudo firewall-cmd --permanent --add-port=15672/tcp
  2. Run this command to open the ports necessary for the cache service. 
    $ sudo firewall-cmd --permanent --add-port=11222/tcp
