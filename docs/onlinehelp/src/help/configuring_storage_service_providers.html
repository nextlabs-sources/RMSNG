Storage service providers configuration

The SkyDRM RMS application enables administrators to configure service providers such as SharePoint and Dropbox, which in turn allow users to add their repositories in SkyDRM RMS and view NextLabs rights-protected and unprotected files.

There are two categories of storage service providers.
  • Configured Providers: This section displays service providers that are configured by the administrator. After the configuration, users can add their own personal repositories for the service provider in SkyDRM RMS.
  • Available Providers: This section displays available service providers that SkyDRM RMS supports. You can configure these service providers to allow users to create repositories in SkyDRM RMS. After the service providers are configured, they appear as Configured Providers. Currently, SkyDRM RMS supports Google Drive, OneDrive, SharePoint Online, SharePoint On Premise, Box, and Dropbox service providers.
