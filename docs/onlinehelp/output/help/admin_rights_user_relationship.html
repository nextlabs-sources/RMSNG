Administrator rights and user role relationship

A table describing the relationship between administrator rights and user roles.

Table 1. Relationship between administrator rights and user roles
Project or system bucket User role Administrator rights Note
Projects Project administrator Yes The project administrator must be a member of the project
Projects Tenant administrator No, unless they are a project administrator too
Projects Normal user No
Projects Super administrator No
Projects API user Yes The API user does not have to be a member of the project
System bucket Tenant administrator Yes
System bucket Normal user No
System bucket Super administrator No
System bucket API user Yes
