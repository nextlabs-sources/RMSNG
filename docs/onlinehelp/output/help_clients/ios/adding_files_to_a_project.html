Adding files to a project

  1. On the Home screen, tap on a project that you are a member of.
    The project page appears.
  2. On the navigation bar, tap .
  3. Tap Add a file.
  4. Tap the location from which you want to add a file.
  5. Select a file to upload.
  6. Tap Done.
    The Create a protected file page appears.
  7. Select the rights protection option to apply to your file, by tapping the slider.
    • View: Enables users to view a rights-protected file. This is selected by default and cannot be disabled.
    • Print: Enables users to print a rights-protected file.
    • Download: Enables users to download a rights-protected file.
    • Watermark: Displays a watermark on a rights-protected file.
    • Validity: defines the expiration of the rights protection for the file.
    Note: The Watermark option is only for users granted with rights.
    If you select Watermark, you can tap Change to define a custom watermark.
    If you select Validity, you can tap Change to define different rights protection validity.
    Alternatively, you can select Company-defined rights.
  8. Tap Create a protected file.
    An upload completion message appears, and the file appears in the Recent Files pane.
