  • Using SkyDRM

    With SkyDRM, you can apply rights protection to a file, and share the rights-protected file.

    • Creating a SkyDRM account

      You can create a SkyDRM account on the SkyDRM app provided by your organization. You should only create an account if your enterprise instructs you to.

    • Logging in to SkyDRM

      You can log in to SkyDRM using your NextLabs SkyDRM, enterprise credentials, or social account credentials.

    • Configuring digital rights preferences

      You can configure digital rights preferences settings to prevent recipients from misusing your sensitive files.

    • Applying rights protection to a file

      You can apply rights protection to files on your local or network drive, MyDrive, or an external repository by email.

    • Sharing a rights-protected file

      You can share a rights-protected file from your local or network drive, MyDrive, or an external repository by email.

    • Marking a file as a favorite

      This section describes the steps to mark a file as a favorite.

    • Making a file available for offline viewing
    • Viewers

      Multiple file formats open in different viewers. This section describes the viewers, common tools, and tools that are specific to each viewer.

      • Common tools

        This section describes the tools common to all 2D viewers. Some tools are not applicable to 3D viewers, such as the CAD viewer.

      • CAD viewer tools

        This section describes the tools specific to the CAD viewer.

    • Changing your password

      If you sign up to SkyDRM directly, without social media or enterprise credentials, you can change your password. Otherwise, you need to change the password where you created it. For example, for Google password, you need to open a Google application and change it from there.

    • Changing your display name

      You can change your display name by following the steps in this section.
