SkyDRM concepts

This section describes the SkyDRM-specific terminology and components, and the relationships between the components of SkyDRM.

MySpace

MySpace is your general repository in SkyDRM. It consists of the following sub-repositories.
  • MyDrive

    MyDrive is the default repository for storing your personal files of any format. Files stored in MyDrive are not rights-protected.

  • MyVault

    MyVault is the default repository for storing your rights-protected (.nxl) files. When you protect your files in SkyDRM, it creates new files with the extension .nxl. The original unprotected files remain in their original location.

Workspace

WorkSpace is the enterprise-level repository, where you can upload files to protect and share with other users within your enterprise. After you upload a file to WorkSpace, the file gets protected, and it becomes a WorkSpace file which no longer has one owner. Other users may use and modify the permissions to the file.

You can create a WorkSpace folder in the SkyDRM Web App or the SkyDRM App for iOS.

Repositories

You can configure SkyDRM to connect to external repositories, such as Box, OneDrive, Dropbox, Google Drive, SharePoint, and SharePoint Online. You can configure a connection to an external repository if your organization allows it.

Project

A SkyDRM project enables you to collaborate with a group of users, on one or more rights-protected documents.

As a project owner, you can create a SkyDRM project, specify the members of the project, and share one or more documents with the project members. Project members can see the documents in the project and upload documents.
